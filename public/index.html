Async and Express App

Assignment 5: A webpage displaying 2 links: one that sends and HTTP request directly to "https://randomuser.me/api/" and another sending a GET request to an EXPRESS server, which then sends acts as an intermediary for the webpage and "https://randomuser.me/api/". The program is run by clicking these links, whose default actions have been prevented, and new event handlers have been programmed to carry out the link's respective duties. Once clicked, the response from the Random Person API will be parsed, and the person's first name, last name, phone number, and email will be displayed below.


  • Send Direct Link to Random Person API.
  • Send Express link to send a request to Express server, which then sends request to Random Person API
